Front-End      Back-End


Front End

The front end is the part of a website or application that users interact with directly. It encompasses everything that users experience visually and functionally in their web browsers.

Key Technologies: 
   HTML (HyperText Markup Language): Structures the content on the web.

    CSS (Cascading Style Sheets): Styles and layouts the HTML elements.

    JavaScript: Adds interactivity and dynamic behavior to web pages.

    React Js:React Js is a free, open‑source, component‑based JavaScript library.


Back-End

The back end is the server-side of a website or application. It manages the database, server, and application logic, ensuring that everything on the client-side actually works.

Key Technologies: 
        Python: Python is a high-level, interpreted, general-purpose language.

        Java: Java is a high-level, object-oriented, platform-independent language.

        Node.js: Node.js is an open-source, cross-platform JavaScript runtime language.

        .NET: .NET is a free, cross-platform, open-source developer platform language.
